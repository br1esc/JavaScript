Facturas

Utilizando las clases de JavaScript, haz un pequeño programa que calcule el precio total de una factura. Para ello:

- Crea una clase Factura que reciba el nombre del cliente, un array de elementos y una pseudoclase información con las propiedades: baseImponible, iva, total y forma de pago

- La clase Cliente recibirá el nombre, dirección, teléfono y nif.

- La clase Elemento recibirá la descripción, cantidad y precio.

Una vez que tenemos las clases deberemos hacer las funciones que van a hacer los cálculos y mostrarlos en la pantalla:

- Cálculo del total: recorreremos los elementos del array de elementos calculando la base imponible de la factura y el total. Esa información la guardaremos en el total de la pseudoclase información.

Base imponible = cantidad * precio.

Total = base imponible * (1+ (100/16))
